Создайте приложение для встреч.
Приложение позволит создавать встречи, удалять их и отмечать как выполненные.

Страница 1 - «Назначения»
 Покажет список предстоящих встреч.
 Каждый элемент в списке будет содержать дату и название встречи, кнопку «удалить» и флажок «завершить». +++
 Удалить появится диалоговое окно подтверждения, и после подтверждения удалите встречу.

Страница 2 - «Назначить встречу»
 Вкладка 1 - «Информация», содержащая следующее:
• Дата - выбор даты (обязательно)                     +++
• Полное имя - текстовое поле (обязательно)           +++
• Примечание - текстовое поле (необязательно)         ---
• Кнопка "Далее"                                      +++
• Кнопка «Назад» (переход на страницу встреч)         +++

 Tab2 - «Подтверждение»
• Детали первой вкладки будут отображаться только для чтения.
• Будет кнопка создания. (сохранит данные и перейдет на страницу встреч)
• Будет иметь кнопку возврата. (перейдет на вкладку Информация)               +++

Общие рекомендации
 Используйте хранилище Vuex с сохранением данных в локальное хранилище (без серверной части)
 Используйте Vue router для навигации
 После перезагрузки - мы должны увидеть данные сохранения
 Вкладки должны быть отдельными компонентами вне компонента родительской страницы.
 Дайте ссылку на репозиторий git с задачей
 Не нужно тратить время на стилизацию, но приложение должно иметь четкий вид.
 Вы можете использовать любые необходимые библиотеки.
